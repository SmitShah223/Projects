# Cancer Prediction Model

This project contains a breast cancer classification workflow implemented in a Jupyter notebook using a tabular diagnostic dataset.

## Contents

- `notebooks/cancer_prediction.ipynb` - exploratory analysis and model development notebook
- `data/data.csv` - input dataset used by the notebook

## Notes

- The notebook appears to focus on binary classification for malignant vs benign diagnosis.
- To make this project more reproducible later, consider adding a `requirements.txt` and an exported model evaluation summary.
