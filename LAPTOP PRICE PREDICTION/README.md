# Laptop Price Prediction

This project contains a notebook-based analysis for predicting laptop prices from tabular product data.

## Contents

- `notebooks/laptop_price_prediction.ipynb` - end-to-end notebook
- `data/laptop_data.csv` - underlying dataset
