# Arduino Security Door

This folder contains materials for a smart door home security system project built around an Arduino-based access and monitoring concept.

## Contents

- `docs/` - project report and supporting documentation
- `media/` - demo video and shared reference link

## Notes

- This project currently contains documentation and media assets rather than source code.
- If you want to expand this project later, a good next step would be to add the Arduino sketch, circuit diagram, and bill of materials.
