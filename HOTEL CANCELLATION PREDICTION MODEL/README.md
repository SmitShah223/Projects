# Hotel Cancellation Prediction Model

This project contains multiple experimental scripts for predicting hotel booking cancellations, along with one generated submission file.

## Contents

- `src/` - model experimentation and preprocessing scripts
- `outputs/` - generated submission artifact

## Notes

- The current repository snapshot does not include the training and test datasets referenced by the scripts.
- Because of that, this project is best treated as an archival experiment bundle unless the original data files are added later.
- A future cleanup pass could consolidate the scripts into a single reproducible training pipeline once the missing datasets are restored.
