# Air Surveillance System AICTE

This project captures the documentation and prototype media for a helium balloon-based IoT surveillance system.

## Contents

- `docs/` - formal project documentation PDF
- `media/` - prototype enclosure and 3D print photos

## Notes

- The folder is organized as a presentation and documentation package.
- A future improvement would be to add firmware, hardware specifications, and system architecture diagrams if they are available.
